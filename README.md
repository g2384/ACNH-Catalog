# ACNH-Catalog

动森助手，一个React Native App

# 安装

1. 安装npm, yarn
2. 在`app`目录下打开terminal（或cmd.exe）
3. 运行`yarn`安装依赖包
4. 运行`yarn android` （mac使用`yarn ios`）

**有任何问题，参见 https://github.com/akveo/kittenTricks/blob/master/DEV_DOCS.md**
