# ACNH-Catalog

动森助手，一个Xamarin App

# 安装

1. 下载&安装 Visual Studio 2019，所有依赖包会自动安装
2. 打开`app\ACNHCatalog.sln`，运行`ACNHCatalog.Android`或`ACNHCatalog.iOS`
